## Important caution

This is a rule-based extraction. It is designed to create a strong first draft, not a final legal chronology. Review Tier 1 entries manually before sending to lawyers, experts, Tower, or a dispute body.

Keep raw `.emlx` files unchanged. Commit processed outputs separately so changes over time are visible.
